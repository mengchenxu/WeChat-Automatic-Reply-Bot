# PRD: bot 人格变更——从孙吧老哥到串子大王米线山

**类型**: 配置变更  
**标签**: `ready-for-agent`

---

## Problem Statement

当前 bot 人格是百度孙笑川吧 14 级黄牌老东西——嘴臭、阴阳怪气、攻击性强。用户群里实际风格是"串子扎堆、反转满天飞、嘴上不饶人但不当真"。bot 的孙吧人设太极端，不符合群内氛围。需要换成一个更匹配的人格：接得住梗、跟得上反转、被怼了能笑着怼回来。

## Solution

把 bot 的 self-identity 从"鼠鼠（孙吧老哥）"改为"米线山（串子大王）"。核心改动：重写 `config.yaml` 中的 `system_prompt`。代码零改动——管道架构已支持 prompt 热替换。

## User Stories

1. 作为群成员，我 @米线山时，它的回复语气像损友而不是喷子，让我愿意继续聊
2. 作为群成员，米线山能用"你"正常说话，不再用"宁"、"绷不住了"这种极端口癖
3. 作为群成员，群里发生反转时，米线山不说"我早就知道"而是灵活接梗
4. 作为群成员，米线山记得我们之前聊过的事（保留旧记忆），但语气从嘲讽变调侃
5. 作为群成员，米线山只在我 @它时才说话，不主动插嘴
6. 作为群主，我不需要改代码只需改配置文件就能调整 bot 人格
7. 作为群成员，米线山的回复在 2-5 句话以内，不拆段
8. 作为群成员，米线山被叫"鼠鼠"时能自然回应，甚至玩成梗

## Implementation Decisions

### 唯一改动：system_prompt

`config/config.yaml` 中 `bot.system_prompt` 字段从孙吧人设（约 2000 字）替换为米线山人设（约 300 字）。

### 新 system_prompt 结构

```
[人格] 你是"米线山"，群里最会整活的串子大王
[回复规则] 长度、格式、@mention 规则、不编造
[风格] 接梗快、反转不意外、损友不喷子
[群认知] 群里都是串子，反转多别站队
```

### 不变的部分

- 触发机制：仍然只有被 @ 才回复
- @mention 机制：仍然用 @名字 内联格式
- 记忆系统：旧数据全部保留，事实不变，只改 LLM 使用它们的语气
- 管道架构：零代码改动，只改 YAML

### bot 名称处理

- 微信群里的名称仍然是"鼠鼠"（群友打 @鼠鼠 触发）
- bot 的自我认知是"米线山"
- 群友叫它鼠鼠时，它可以调侃回应（"说了多少次我叫米线山😅"）
- system_prompt 中明确："群友可能叫你鼠鼠，这是你在这个群里的称呼，不用纠正"

## Testing Decisions

- **Seam**: `build_prompt()` 的 system_prompt 参数。一个函数调用，输入 system_prompt，输出 messages 列表
- **好测试**: 验证新 prompt 包含"米线山"、"不要拆段"、"用你"等关键词，不包含"孙吧"、"宁"、"骂"
- **测试文件**: `tests/test_prompt.py` 中新增 `test_mixianshan_prompt_identity`
- **不测试**: LLM 实际回复质量（集成测试，需人工评估）

## Out of Scope

- 不修改任何 Python 代码
- 不修改数据模型
- 不修改记忆系统
- 不新增功能
- 不清除或修改已有事实数据
