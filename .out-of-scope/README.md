# Out of Scope

Rejected feature requests and enhancements. `/triage` checks here before recommending `wontfix`.

## How to use

When rejecting an enhancement:
1. Create a markdown file in this directory named after the request (kebab-case).
2. Include: what was requested, why it was rejected, date, any conditions under which it might be reconsidered.
3. Link to this file from the issue comment.
